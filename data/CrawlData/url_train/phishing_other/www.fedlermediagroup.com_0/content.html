Security Basics
  1. We will NEVER ask you for your Internet Banking
    Password or Security Token Passwords via electronic
    message or telephone.
  2. ALWAYS keep your Username, Password and Security
    Token Passwords secret.
  3. NEVER use a link or an attachment in any message to
    access Internet Banking.
  4. ALWAYS check that the website address bar and
    certificate both match capitecbank.co.za.
  5. Monitor your accounts closely and report any
    suspicious activity immediately to 0860 10 20 43.
